# Sample Configuration Files

Sample configuration and data files for testing the Autograder system.

## Files

### criteria.json
Empty sample criteria file. See `../criteria_examples/` for complete examples.

### feedback.json
Sample feedback configuration showing how to structure feedback messages.

### setup.json
Sample setup configuration for container initialization (runtime, commands, etc.).

### requirements.txt
Sample Python requirements file for dependencies.

## Usage

These files serve as templates or test data for:
- Manual testing of the autograder
- Creating new configurations
- Understanding the expected file formats

For complete, working examples, see:
- **Criteria examples**: `../criteria_examples/`
- **Code examples**: `../code_examples/`
