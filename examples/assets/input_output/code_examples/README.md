# Code Examples

Sample code submissions in multiple programming languages for testing the Input/Output template.

## Languages

- **Python** - python/
- **Java** - java/
- **JavaScript** - javascript/
- **C++** - cpp/

## Calculator Examples

Each language directory contains calculator implementations with varying levels of functionality:

### Simple Calculator
Basic calculator with addition operation only.

### Standard Calculator
Complete calculator with:
- Addition
- Subtraction
- Multiplication
- Division

### Advanced Calculator
Full-featured calculator including:
- Basic operations (add, subtract, multiply, divide)
- Advanced operations (power, modulo, square root)
- Error handling
- Edge case support

### Broken Calculator (Optional)
Intentionally buggy implementations for testing failure scenarios.

## Usage

1. Load code examples in the testing dashboard
2. Select a language and example
3. Submit to the autograder with appropriate criteria
4. Review grading results and feedback

## Testing

Use these examples with the criteria examples in `../criteria_examples/`:
- Simple criteria for basic examples
- Complex criteria for advanced examples
- Test bonus/penalty scoring with different implementations
