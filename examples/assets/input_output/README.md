# Input/Output Template Assets
Example files for the Input/Output template.
## Contents
- **code_examples/**: Sample code in Python, Java, JavaScript, C++
- **criteria_examples/**: 5 pre-built criteria templates (simple to complex)
- **sample_files/**: Configuration file examples
- **dashboard/**: Old dashboard UI (deprecated - use `/examples` instead)
## Criteria Templates
1. Base Only - Simple test suite
2. Base + Bonus - With extra credit
3. Base + Bonus + Penalty - Full scoring
4. With Subjects - Organized by topics
5. Nested Subjects - Hierarchical structure
