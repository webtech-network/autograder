# Example Assets
Sample files and test data for autograder templates.
## Structure
```
assets/
├── input_output/       # I/O template examples
│   ├── code_examples/  # Python, Java, JS, C++ samples
│   ├── criteria_examples/  # 5 criteria templates
│   ├── sample_files/   # Config files
│   └── dashboard/      # Old UI (deprecated)
├── web_dev/           # Web dev template examples
└── api_testing/       # API testing template examples
```
## Usage
Example files are referenced by the interactive demo. Code examples can be used as reference for submissions.
