# Security Policy

## Supported Versions

Security fixes are currently prioritized for the latest code on the `main` branch.

## Reporting a Vulnerability

Please **do not** open public GitHub issues for security vulnerabilities.

Report vulnerabilities privately by email:

- arthurcarvalhorodrigues2409@gmail.com

Include as much detail as possible:

- A clear description of the issue
- Reproduction steps or proof of concept
- Potential impact
- Any suggested mitigation

We will review reports and coordinate a fix and disclosure process.
