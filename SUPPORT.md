# Support

Use the right channel so maintainers can help effectively:

- **Questions / Discussions**: [GitHub Discussions](https://github.com/webtech-network/autograder/discussions)
- **Bug Reports / Feature Requests**: [GitHub Issues](https://github.com/webtech-network/autograder/issues) (use issue templates)
- **Security Reports**: Follow [SECURITY.md](SECURITY.md) and report privately by email

If you want to contribute code, follow [CONTRIBUTING.md](CONTRIBUTING.md).
